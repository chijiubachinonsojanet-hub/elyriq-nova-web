How I Built and Published My First Website: A Beginner-Friendly Guide

Want to learn how I built and published the Elyriq Nova website from scratch? This guide explains exactly what I did in simple steps — even if you’ve never built a website before.

Step 1: Set Up Your Workspace

First, create a folder where your website files will live.

You can do this in two ways:

Option A: Using VS Code
Open VS Code.
Create a new folder.
Name it something like "my website ai".
Option B: Using Your Computer/Desktop
Go to your Desktop or preferred location.
Right-click and create a New Folder.
Name it "my website ai".
Right-click the folder and select Open with Code (if VS Code is installed).

Inside your folder, create these two files:

index.html → This controls the website content and structure.
style.css → This controls the design, colors, and styling.
Step 2: Build Your Website

Now it’s time to create the website.

I added the website content, including:

Brand information
Text and headings
Colors and styling
Images and layout details

You can:

Write the code yourself, or
Use AI tools to help you generate or improve your code.

When you finish making changes, remember to save your files.

Step 3: Preview and Review Your Website

Before publishing your website, check how it looks.

Go to your project folder.
Double-click the index.html file.

Your website should open in your browser.

Review it carefully:

Does everything look correct?
Are the colors, images, and text displaying properly?
Do you need to make any adjustments?

Make edits if needed and save again.

Step 4: Publish Your Website Online Using GitHub

Once your website looks good, you can make it live on the internet using GitHub.

Create a Repository
Log into your GitHub account.
Click New Repository.
Give your repository a name.
Set It to Public

Select Public so your website can be hosted for free.

Upload Your Files

Upload your project files, such as:

index.html
style.css
janet.jpeg
victoria.jpeg
Enable GitHub Pages
Open your repository.
Go to Settings.
Click Pages.
Under Branch, select main.
Under Folder, choose root.
Click Save.

GitHub will automatically generate a public website link for you.

Final Result

Congratulations!

By following these steps, I successfully built and deployed my website to the cloud using GitHub Pages.

My website is now live and accessible online for anyone to view.

If you're a beginner, you can follow this same process to build and publish your own website too.
